# Sales & Demand Forecasting - Task 1

## Overview
This project builds a machine learning model to forecast future business sales based on historical data using Linear Regression.

## Contents
- `notebook.ipynb`: Contains the Python code, data visualization, and model predictions.

## Key Learnings

- Implemented a Linear Regression model using Scikit-learn.
- Visualized actual vs. predicted sales trends.
